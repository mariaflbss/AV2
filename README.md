# Aerocode-AV2

## Pré-requisitos

- Node.js
  
## Como rodar o projeto

1. Clone o repositório ou faça o download do projeto.

2. Abra o terminal e navegue até a pasta do projeto:

cd aerocode-av2

3. Instale as dependências do projeto:
   
npm install

4. Inicie o servidor:
 
npm run dev

5. Abra seu navegador e acesse:

http://localhost:5173
